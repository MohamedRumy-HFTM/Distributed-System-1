# Distributed System 1

Dieses Repository enthaelt das erste Quarkus-Projekt im Ordner `code-with-quarkus`.

## Projekt starten

```bash
cd code-with-quarkus
./mvnw quarkus:dev
```

## Endpunkte

Nach dem Start sind diese URLs verfuegbar:

- <http://localhost:8080/hello>
- <http://localhost:8080/blog>
- <http://localhost:8080/q/dev/>

Der Endpunkt `/blog` gibt folgenden Text zurueck:

```text
Willkommen auf meinem Blog!
```
